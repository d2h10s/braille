# Braille Printer PC SW
## Brief
이 프로그램은 자체 제작한 점자 프린터의 HW를 지원하는 PC SW입니다.<br>
텍스트 편집, 자동 한글->점자 변환기, TTS, OCR 기능이 포함되어 있습니다.<br>
<br>
점자 프린터는 ARM Cortex-M4 프로세서가 사용 되었으며, SW는 mbed-os 2.0을 기반으로 제작되었습니다.<br>
